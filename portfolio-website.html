Machine Elements Analysis Image

Basic Machine Elements Course Project

Conducted structural analysis of Honda Vario 150 components using SolidWorks and theoretical mechanics.

Show details ↓
Hide details ↑

For this academic project, I conducted a comprehensive structural analysis of three Honda Vario 150 components: the side stand spring, main stand shaft, and kickstarter flange bolt. The project involved several key phases:

First, I created detailed CAD models and engineering drawings of each component using SolidWorks, accurately capturing their dimensions and mechanical features. This required careful measurement and application of technical drawing principles.

Next, I performed theoretical stress analysis on each component under their respective operating conditions. For the side stand spring, I applied fatigue analysis methodologies including Gerber and Goodman criteria to identify critical stress regions. The main stand shaft analysis involved creating moment and shear force diagrams to locate critical points, while the flange bolt required evaluation of different failure modes including yielding, joint separation, and overload conditions.

The project output included a comprehensive report documenting the analysis process and conclusions about each component's safety under operational loading. The calculations confirmed that all three components were adequately designed with appropriate safety factors, validating the manufacturer's design decisions from an engineering perspective.
